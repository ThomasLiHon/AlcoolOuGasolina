⛽ Calculadora Álcool ou Gasolina
Projeto simples feito com HTML, CSS e JavaScript para calcular qual combustível compensa mais.

🚀 Como funciona
Se:

Preço do Álcool ÷ Preço da Gasolina ≤ 0.7

➡ Compensa abastecer com Álcool

Caso contrário

➡ Compensa abastecer com Gasolina
